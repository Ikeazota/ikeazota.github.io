About Me.

Augustine Ikeazota is an IT Support Engineer at QShelter Limited, with over three years of experience in the industry. He excels in software and hardware troubleshooting, system configurations, user training and support, network diagnostics, and overall IT user support. Throughout his career, Augustine has successfully executed several key technological projects, including the configuration and administration of Odoo CRM ERP solutions, providing comprehensive user training, and managing the seamless migration of 22 users from Google Workspace to Microsoft 365..

Augustine's skill set encompasses Microsoft 365 Administration, Google Workspace management, Odoo ERP administration, system administration, Application installation and troubleshooting, network configuration, technical issue diagnostics and Python programming. His passion for IT drives him to continually expand his expertise. Currently, Augustine is embarking on an exciting journey into Cloud and DevOps Engineering.
